# Walmart Retail Sales Analysis

This project analyzes Walmart's weekly sales data to identify seasonal patterns and build predictive models for sales performance.

## Dataset
The dataset comes from the Walmart Kaggle dataset and includes:
- Weekly sales data
- Store information
- Economic indicators

Total dataset size: ~420,000 records.

## Research Questions

1. How do weekly sales vary by month?
2. Are holiday weeks associated with higher sales?
3. Is unemployment related to weekly retail sales?

## Methods Used

- Data cleaning and feature engineering
- Exploratory data analysis
- Welch two-sample t-test
- Polynomial regression
- Logistic Regression
- Random Forest
- Naive Bayes classification

## Key Findings

- Sales show strong seasonality with peaks in November and December.
- Holiday weeks have significantly higher average sales.
- Unemployment has a weak relationship with weekly sales.
- Logistic Regression achieved the best classification performance (AUC ≈ 0.59).
